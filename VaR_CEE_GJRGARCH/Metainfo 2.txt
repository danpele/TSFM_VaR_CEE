Name of Quantlet: VaR_CEE_GJRGARCH

Published in: 'Zero-Shot Foundation Models for VaR and ES Forecasting in CEE Markets'

Description: Implements GJR-GARCH(1,1) with skewed-t distribution for VaR and ES forecasting using a rolling 250-day window with daily refitting. Falls back to normal distribution if skewed-t fails to converge.

Keywords: GJR-GARCH, volatility modeling, Value-at-Risk, Expected Shortfall, skewed-t distribution, leverage effect, rolling window

Author: Daniel Traian Pele

Submitted: 15 April 2026

Datafile: ../data/processed/{country}_returns.parquet

Input: Log return series, rolling window of 250 days, VaR levels (1%, 2.5%, 5%), out-of-sample start date

Output: CSV files with daily VaR and ES forecasts per series in ../data/var_forecasts/

Example: Fits GJR-GARCH(1,1)-skewt daily on rolling 250-day window for WIG20_ret and forecasts 1-day VaR
